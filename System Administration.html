System Administration

  • iptables (防火牆設定) 
        /etc/sysconfig/iptables 
    /etc/init.d/iptables restart // 重新 reload 防火牆設定
  • ssh 連道遠端主機執行 process 後, 斷線 process 還要存活 
        nohup ./whatever > /dev/null 2>&1 &
  • 查看 TCP 被什麼 process 佔用 
        netstat -tln # 可以看到有哪些 IP:Port 開啟
    netstat -plten | grep java
  • 查看 Linux TCP Port 被哪隻程式(Process)佔用, 可以用下述的命令: 
        sudo lsof -i
    sudo lsof -i | grep TCP
    sudo lsof -i :80 | grep LISTEN
    sudo netstat -lptu
    sudo netstat -tulpn
    sudo ls -l /proc/$pid/exe
  • 列出此 TCP Port 目前有哪些 PID 
        sudo fuser 80/tcp # 會把目前使用此 Port 的 PID 全部列出
  • 關於使用者權限 
        // 列出系統所有的 group
    cat /etc/group
    
    // 將某個 user 加入 root 的 group
    sudo adduser browny root
    
    // 查詢某個 user 屬於哪些 group
    groups browny
  • 查看硬碟使用量 
        du -h <directory>
    du -hcs <directory> // summary
  • 查看 httpd status 
        啟動 mod_status
