Installation (Ubuntu 12.04)

1. Install dependencies

sudo apt-get install erlang erlang-src gnuplot perl libtemplate-perl

2. Download Tsung

http://tsung.erlang-projects.org/dist/

3. Install Tsung

tar -zxvf tsung-1.4.2.tar.gz
cd tsung-1.4.2/
./configure
make
sudo make install

4. Verify installation

Use command tsung, check if below message shown
Usage: tsung <options> start|stop|debug|status

5. The default testing config will be placed in

/usr/share/doc/tsung/examples

Report

Run /usr/lib/tsung/bin/tsung_stats.pl in log folder then firefox graph.html

Issues

1. 跑 register loading test 的時候在 openfire admin console 沒看到 user 增加?

Ans; Server Settings -> Registration Settings -> Inband Account Registration -> Set to Enabled

2. 要 register 2000 個, 結果只建立了 5 個

Ans: 先修改系統的最大數限制 

	在本機 /etc/security/limits.conf 最後增加：
	* soft nofile 65535
	* hard nofile 65535
	* soft nproc 65535
	* hard nproc 65535
	
	在配置文件中配置一個重要的參數 maxusers
	<clients>  
	    <client host="localhost" use_controller_vm="true" maxusers="60000">  
	    </client>  
	</clients>
