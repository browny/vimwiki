Ruby

Installation

  • 使用 RVM install ruby

https://rvm.io/ 

    \curl -L https://get.rvm.io | bash -s stable
    
    // 列出 RVM 上可安裝的 ruby version
    rvm list known
    
    // 安裝某版本的 ruby
    rvm install 1.9.3
    rvm use 1.9.3
    rvm rubygems latest
    
    // 全域的安裝動作
    rvm @global do gem install <xxx>
