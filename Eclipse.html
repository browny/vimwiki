Eclipse hotkey

1.  Ctrl + Shift + O     // Organize imports
2.  Ctrl + /             // Line Comment
3.  F3                   // Open Declaration
4.  Ctrl + Shift + F     // Auto indent
5.  Alt + Shift + S      // Open Source menu
6.  Alt + /              // Content Assist
7.  Ctrl + D             // Delete row
7.  Alt + <-             // Back to previous tab
8.  Ctrl + Alt + H       // Open call hierachy
9.  Ctrl + Shift + Y     // 小寫字母
10. Ctrl + Shift + X     // 大寫字母
11. Shift + Ctrl + R     // Open Resource
12. Shift + Ctrl + T     // Open Type
13. ALT + Up/Down Arrow  // Move the row (or the entire selection) up or down
14. Ctrl + 1             // Quick fix
15. Ctrl + E             // Shows you a list of all open editors
16. Ctrl + Shift + L     // Shows you a list of your currently defined shortcut keys.
17. Ctrl + PageUp/Down   // Switch between tabs
18. Alt + Shift + A      // Toggle vertical selection
19. Ctrl + O             // 快速打開 class 當中的 method 選單

eclim

Step1. 

    // 安裝 eclim
    java -jar eclim_2.2.5.jar

Step2. 

    // 啟動 eclim deamon
    /opt/eclipse/eclimd start

Step3. 

    // 編輯 ~/.eclimrc, 加入你的 workspace 位置
    osgi.instance.area.default=@user.home/myworkspace

Step4. 

    // 查看 eclimd 是否正確運行
    :PingEclim
	
    // 開 vim, :ProjectList 看目前有哪些 project
    :EclimEnable 
    <Ctrl+X><Ctrl+U> 跳出提示

Tips

  • 可以把 Validation 的大部分選項關掉, 留下 Classpath Dependency Validator 就可以了
  • Eclipse 如果沒開 Build Automatically, 會發生 class 沒有被 build 出來的情形，可以打開隱藏資料夾看, 並且有編輯 code 以後, server 視窗應該會從 Synchronized 變成 Republished, 這樣才是正確。或者可以自己加 log 看看有沒有真的 build 新的出來
  • Q: 明明 .java 有在 package 的路徑下，但是跳出找不到 package 的 error: The declared package does not match the expected package
    A; 把 .java 移出資料夾，F5 更新 Eclipse，再把 .java 移回去，再按 F5
