Eclipse hotkey
1. Ctrl + Shift + O     // Organize imports
2. Ctrl + /             // Line Comment
3. F3                   // Open Declaratio
4. Ctrl + Shift + F     // Auto indent
5. Alt + Shift + S      // Open Source menu
6. Alt + /              // Content Assist
7. Ctrl + D             // Delete line
Tips

1. 可以把 Validation 的大部分選項關掉, 留下 Classpath Dependency Validator 就可以了

2. Eclipse 如果沒開 Build Automatically, 會發生 class 沒有被 build 出來的情形，可以打開隱藏資料夾看, 並且有編輯 code 以後, server 視窗應該會從 Synchronized 變成 Republished, 這樣才是正確。或者可以自己加 log 看看有沒有真的 build 新的出來
