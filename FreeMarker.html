Usages

  • Put ${variable} in .ftl template file for the word you want to replace
  • In code, use Map to assign real value for the variable 
	Map<String, Object> map = new HashMap<String, Object>();
	map.put("user", name);
	
	private Template welcomeLetterTmpl;
	Writer out = new StringWriter();
	welcomeLetterTmpl.process(map, out);

Issues

1. How to load resource?

  • put the xxx.ftl under src/main/resources/template in maven project
  • use setClassForTemplateLoading as below, the path inner freemarkerConfiguration.getTemplate is the path inner src/main/resources/ 
	freemarkerConfiguration = new Configuration();
	freemarkerConfiguration.setClassForTemplateLoading(this.getClass(), "/");
	Template freemarkerTemplate = freemarkerConfiguration.getTemplate("template/vendor.tpl");
