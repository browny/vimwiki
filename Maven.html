4. Maven build
mvn compile, mvn package: 這種後面沒有冒號的叫做 life cycle, 其背後實作就是去一連串 goals 的呼叫
mvn tomcat6:run: 這種後面有冒號的叫做 plugin, 這個例子就是呼叫 tomcat6 這個 plugin 裡面的 run 這個 goal
mvn help:describe: 這是 mvn 的 help 查詢 plugin 使用方法
mvn tomcat6:run -P integration-server: 載入 integration-server 這個 profile

5. pom.xml 設定 resource 路徑 

InputStream is = getClass().getResourceAsStream("/application.properties")
Properties urbanProperties = new Properties();
urbanProperties.load(is);

因為 resource 有分用來 filtering 跟非 filtering 兩種, 如果只用第一個的話就只會包含到 application.properties 和 logback.xml, 所以還需要第二個 filtering 屬性為 false 的宣告 


<resources>
	<resource>
		<directory>src/main/resources</directory>
		<filtering>true</filtering>
		<includes>
			<include>application.properties</include>
			<include>logback.xml</include>
		</includes>
	</resource>
	<resource>
		<directory>src/main/resources</directory>
		<filtering>false</filtering>
		<excludes>
			<exclude>application.properties</exclude>
			<exclude>logback.xml</exclude>
			<exclude>*.swp</exclude>
			<exclude>*.swo</exclude>
		</excludes>
	</resource>
</resources>
